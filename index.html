Introduction to Backend Web Development

The blogs below will help you learn the basics of backend web development by helping you to create a simple web server (using express framework) and then connect it to a database (MongoDB).

Along the way, you will also learn the basic CRUD operations.That is, you will learn how to post data to the database from the server. Then you will learn how to read, edit and delete the data you posted (or added) earlier.

While learning the basic CRUD, you will also learn about APIs which are used to implement the CRUD operations.

Note: It is best to read the blogs in the order they have been written.

1. Creating a simple server using express

2. Serving a message to the browser

3. Connecting express with local mongodb

4. Pushing data from express server to mongodb

5. Server Routing Basics Using JSON Server

6. Sending Data from Frontend to Backend Via HTML Form
